This library is for controlling Dynamixel AX-12 servo motors.  

It implements the protocol described here: https://emanual.robotis.com/docs/en/dxl/protocol1/  

It expects that the rx/tx lines of whichever UART bus is used are converted to half-duplex in hardware e.g. as shown in the documentation here (botton of page) https://emanual.robotis.com/docs/en/dxl/ax/ax-12a/
